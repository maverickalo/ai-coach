# Coach AI

A simple, mobile-friendly static website for Coach AI with public Privacy
Policy and Terms & Conditions pages suitable for Twilio A2P campaign
registration.

## Files

- `index.html` - Coach AI overview
- `privacy.html` - Privacy Policy and SMS disclosures
- `terms.html` - Terms & Conditions and SMS consent terms

## Enable GitHub Pages

1. Push this repository to GitHub with the site files on the `main` branch.
2. Open the repository on GitHub.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select the **main** branch and the **/ (root)** folder.
6. Click **Save**.

GitHub will publish the site and display its public URL on the Pages settings
screen.
