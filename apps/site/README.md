# Coach AI

A simple, mobile-friendly static website for Coach AI with public Privacy
Policy and Terms & Conditions pages suitable for Twilio A2P campaign
registration.

## Files

- `index.html` - Coach AI overview
- `privacy.html` - Privacy Policy and SMS disclosures
- `terms.html` - Terms & Conditions and SMS consent terms

## GitHub Pages

The root-level HTML files mirror this directory so the existing public routes
remain stable. In GitHub, use **Settings → Pages → Deploy from a branch →
main → / (root)**.
